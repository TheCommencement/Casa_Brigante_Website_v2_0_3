# Casa Brigantè · Website v1.0

The public home of The Commencement. Static site: no build step, no dependencies.

## Run it
1. Open this folder in VS Code.
2. Right-click `index.html` → "Open with Live Server."

## Work on it with Claude Code
Open this folder in VS Code, open the terminal, and run `claude`. It reads `CLAUDE.md` automatically, which carries the voice rules, banned words, and design laws, so every session enforces the constitution.

Good first prompts:
- "Add the Amazon link for [title] to the Take a copy button and the catalog card."
- "Build books.html using the existing tokens and voice, one page for the full catalog."
- "Swap the Studio card image to assets/images/pimp-chronicles.jpg and update the alt text."

## Files
- `index.html`: homepage v1.0
- `404.html`: in the voice of the house
- `styles.css`: all design tokens at the top; never add values outside them
- `app.js`: mobile nav, scroll reveals, correspondence form
- `CLAUDE.md`: the constitution; read it before changing anything
- `assets/`: covers, artist images, brand marks

## Deliberately unfinished
See the "Pending" list in CLAUDE.md. Nothing there is forgotten; it is waiting on real assets and live links.
